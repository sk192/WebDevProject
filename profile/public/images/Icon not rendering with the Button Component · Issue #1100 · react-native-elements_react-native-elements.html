Versions:

"react-native": "0.55.1",
"react-native-elements": "^0.19.1",

Description

  1. Explain what you did
    Trying to use the Button component with an Icon.

  2. What you expected to happen
    The Button component should have a vector icon along with the text.

  3. What actually happens
    Button with text but no Icon is appearing.

I thought this might be because react-native-vector-icons might not have installed properly but this is not the case as I'm able to use the vector icons when not using with the Button component of this library.

Reproduction Steps and Sample Code

Here is the code snippet:

import React from "react";
import { View, Text, StyleSheet } from "react-native";
import { Button } from "react-native-elements";
import Icon from "react-native-vector-icons/FontAwesome";

const Home = () => (
  <View style={styles.container}>
    <Button
      icon={<Icon name="arrow-right" size={30} color="#000" />}
      title="BUTTON WITH ICON"
      buttonStyle={{
        backgroundColor: "rgba(92, 99,216, 1)"
      }}
    />
    <Icon.Button name="facebook" backgroundColor="#3b5998">
      Login with Facebook
    </Icon.Button>
  </View>
);

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: "center"
  }
});

export default Home;

Hey @vikrantnegi
If you use v0.19.x, the Button component takes an object:

<Button
  icon={{ name: 'arrow-right', type: 'font-awesome' }}
  title="BUTTON WITH ICON"
  buttonStyle={{
    backgroundColor: "rgba(92, 99,216, 1)"
  }}
/>

If you use the last version v1.0.0-beta4, the Button has been rewritten from scratch, and you should do:

<Button
  icon={<Icon name="arrow-right" color="white" size={18} />}
  title="BUTTON WITH ICON"
  buttonStyle={{
    backgroundColor: "rgba(92, 99,216, 1)"
  }}
/>

Remember that on the documentation website, you can see the versioned doc by clicking on the version on the header
